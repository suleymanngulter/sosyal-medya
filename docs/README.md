# 📚 Dokümantasyon

Bu klasör, sosyal medya uygulamasının tüm dokümantasyonunu içerir.

## 📖 Dokümantasyon Listesi

### [ARCHITECTURE.md](./ARCHITECTURE.md)
Proje mimarisi, klasör yapısı, tasarım desenleri ve genel yapı hakkında detaylı bilgi.

**İçerik:**
- Genel mimari yapı
- Klasör yapısı
- Veri akışı
- Tasarım desenleri
- Güvenlik
- Performans optimizasyonları

### [API_INTEGRATION.md](./API_INTEGRATION.md)
Backend API entegrasyonu için detaylı rehber.

**İçerik:**
- API yapısı
- Entegrasyon adımları
- API endpoint'leri
- Örnek kullanım
- Troubleshooting

### [TESTING.md](./TESTING.md)
Test rehberi ve troubleshooting bilgileri.

**İçerik:**
- Test adımları
- Debug ipuçları
- Yaygın sorunlar ve çözümler
- Test checklist

## 🚀 Hızlı Başlangıç

1. **Yeni başlıyorsanız:** [ARCHITECTURE.md](./ARCHITECTURE.md) dosyasından başlayın
2. **API entegrasyonu yapıyorsanız:** [API_INTEGRATION.md](./API_INTEGRATION.md) dosyasını okuyun
3. **Test ediyorsanız:** [TESTING.md](./TESTING.md) dosyasına bakın

## 📝 Dokümantasyon Güncelleme

Dokümantasyon güncellemeleri için:
1. İlgili `.md` dosyasını düzenleyin
2. Markdown formatına uygun yazın
3. Commit mesajında "docs:" prefix'i kullanın
